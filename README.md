 Superheroes 30 Days App
📌 Project Overview

Superheroes 30 Days is an Android application built using Kotlin and Jetpack Compose.
The app displays a list of superheroes in a beautifully designed scrollable layout, where each day highlights a different hero with their image, name, and description.

This project demonstrates modern Android development practices using Material 3 design principles.

🚀 Features

📅 30 Days of Superheroes concept

📜 Scrollable list using LazyColumn

🖼 Hero image display

📝 Name and description for each hero

🎨 Material 3 Theming (Color, Typography, Shape system)

🏗 Clean architecture with separate UI and Model layers

🛠 Tech Stack

Kotlin

Jetpack Compose

Material 3

Android Studio

MV-style structure (Model + UI separation)
